# License Information

This project uses a dual-license structure to appropriately cover different types of content:

## License Mapping

| Content Type | License | File |
|--------------|---------|------|
| **Code** (HTML, CSS, JavaScript, configuration files) | MIT License | [LICENSE](./LICENSE) |
| **Content** (Text, images, datasets, documentation) | Creative Commons Attribution 4.0 International (CC BY 4.0) | [LICENSE-CC-BY-4.0.txt](./LICENSE-CC-BY-4.0.txt) |

## Attribution Requirement

All uses of code or content from this project must include the following attribution:

**Attribution: Ramon Logan Jr.**

### Placement

- In documentation: Include in an acknowledgments or attribution section
- On websites: Include in the footer or credits page
- In derived works: Maintain attribution in visible/accessible locations

## File-Level Header Format

```markdown
<!--
```

## Quick Reference

- **MIT License**: Permissive software license allowing commercial use, modification, distribution, and private use
- **CC BY 4.0**: Content license requiring attribution, allowing commercial use, modification, and sharing
- **Attribution Line**: `Attribution: Ramon Logan Jr.`
